# Number Guessing Game (Java)

## Description
This is a Java console-based Number Guessing Game where the program randomly
generates a number between 1 and 150. The user has 5 attempts to guess the
correct number with hints provided after each attempt.

## Features
- Random number generation
- Limited guessing attempts
- User-friendly hints
- Console-based interaction

## Technologies Used
- Java
- Scanner class
- Command Line Interface (CLI)

## How to Run
1. Install Java JDK
2. Open terminal or command prompt
3. Navigate to the project directory
4. Compile:
   javac NUMBER_GUESSING.java
5. Run:
   java NUMBER_GUESSING

## Sample Output
A number is chosen between 1 and 150.
You have 5 attempts to guess the correct number.
